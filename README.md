# 🚗 Car Showroom Management System

A web-based application designed to automate and manage showroom operations such as vehicle inventory, customer records, and booking management.

---

## 📌 Project Overview

The **Car Showroom Management System** is developed to simplify and digitize the workflow of a car showroom. It helps manage vehicles, customers, and bookings efficiently through an interactive web interface.

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* PHP
* MySQL
* XAMPP (for local server and database)

---


## 🚀 How to Run the Project

1. Install **XAMPP**
2. Start **MySQL**
3. Place the project folder inside:

   ```
   htdocs/
   ```
4. Import the database file into **phpMyAdmin**
5. Open browser and run:

   ```
   http://localhost/your-folder-name
   ```

---

## 🎯 Purpose

This project is developed to:

* Automate showroom operations
* Improve efficiency and data management
* Provide a user-friendly interface for both admin and customers
